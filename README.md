# Contador de Visitas Web

## Descripción
Aplicación de consola en Java que registra y cuenta las visitas a páginas web utilizando un HashMap.

## Funcionalidades
- Registrar páginas web visitadas
- Contar automáticamente el número de visitas por página
- Mostrar el listado completo con el recuento de visitas

## Tecnologías
- Java
- HashMap
- Scanner
